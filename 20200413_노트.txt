분기문, 반복문 노트 - 20200418

기본 사용방법

if(조건식){
	조건에 맞을 때 수행할 내용
}else{
	조건에 맞지 않을 때 수행 할 내용(else문 조건에서 수행 할 내용이 없으면 else문을 생략 가능)
}

switch(조건식){ switch문은 if문과 유사하나 대부분의 case문은 if문으로 변경이 불가능.
	case(조건):
		break; // 생략은 가능함, 하지만 생략하면 해당 조건문 아래에 있는 조건을 전부 실행하게 됨.
	case(조건):
		break;
	case(조건):
		break;
	case(조건):
		break; switch문의 마지막 행은 break생략하여도 추가로 실행할 조건이 없어서 문제 X
}

위 두가지 사항에서 조건식은 비교연산이다.

if, switch문의 기본적 특징은 프로그램의 흐름을 제어하는 조건문이다.              

if, switch문은 JDK 1.7 이후부터는 String 도 비교 가능하다. 그 이전은 불가능.
